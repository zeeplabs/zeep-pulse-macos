# Zeep Pulse — Releases

Signed, notarized macOS builds and the Sparkle auto-update feed (`appcast.xml`) for [Zeep Pulse](https://github.com/zeeplabs/zeep-pulse).

Source code lives in the private `zeeplabs/zeep-pulse` repo — this repo only holds public release binaries and the update feed, since Sparkle (running unauthenticated on end users' Macs) can't read a private repo.
